# P12199 Hash Killer III 解题说明

## 问题背景

Hash Killer III 是洛谷的一个开放性问题，主要考察对哈希函数（特别是多项式滚动哈希）的理解和攻击方法。

## 常见哈希函数

在竞赛中常用的字符串哈希函数为：

```
hash(s) = (s[0] * BASE^(n-1) + s[1] * BASE^(n-2) + ... + s[n-1]) mod MOD
```

其中常见参数：
- BASE = 131 或 13331
- MOD = 1e9+7 或其他大质数

## 攻击策略

### 1. 数学构造法
通过数学计算构造两个不同的字符串使其哈希值相同。
原理：如果 `(c1-c2)*BASE^k ≡ (c3-c4)*BASE^m (mod MOD)`，则可以构造碰撞。

### 2. 最坏情况生成
生成大量字符串使它们在哈希表中产生大量碰撞，导致性能退化。

### 3. 生日攻击
利用生日悖论，生成大量随机字符串，概率性地找到哈希碰撞。
当生成 O(√MOD) 个字符串时，有较高概率找到碰撞。

### 4. 周期性利用
利用 `BASE^k mod MOD` 的周期性构造碰撞。

## 实现说明

本实现提供了多种策略的函数：
- `generateMathematicalCollision()`: 数学构造法
- `generateWorstCase()`: 最坏情况生成
- `generateBirthdayAttack()`: 生日攻击
- `generatePeriodicCollision()`: 周期性攻击

主函数 `solve()` 使用了基于 BASE 的模式生成，可以根据具体题目要求调用不同的函数。

```bash
echo "10" | ./p12199
```
